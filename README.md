## Команды по развертыванию приложения

- composer install
- php artisan migrate
- php import:book
- php artisan serve
- npm install
- npm run dev

## Через докер
- composer install
- ./vendor/bin/sail up -d
- ./vendor/bin/sail artisan migrate
- ./vendor/bin/sail import:book
- npm install
- npm run dev
